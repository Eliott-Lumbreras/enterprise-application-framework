# Design docs (Fase 10 — AI Architect)

Un archivo por módulo: `<kebab-case>.design.md`, generado a partir de
`.claude/templates/design-doc.template.md` y llenado siguiendo
`.claude/skills/ai-architect.md` antes de correr
`scripts/generate-module.js` para ese módulo.

Validar con:

```bash
node scripts/check-design.js <nombre-modulo-kebab>
```

Ver `.claude/checklists/design.checklist.md` para el detalle de qué debe
contener cada sección.
