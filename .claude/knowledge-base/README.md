# Knowledge base (Fase 7)

A diferencia de los skills (Fase 2, reglas genericas de como programar), esta carpeta guarda conocimiento especifico del negocio minero real con el que se ha trabajado (proyecto "Aura Turno" / Fast2Mine), para que Claude no tenga que redescubrirlo cada vez.

Regla estricta: todo lo marcado como "confirmado" viene de datos/contratos reales verificados en el proyecto Aura Turno. Todo lo marcado como "pendiente" es una hipotesis razonable basada en terminologia minera estandar, NO un dato verificado — no se debe usar como si fuera definitivo hasta confirmarlo con el usuario o el proveedor de datos.

| Archivo | Contenido |
|---|---|
| glossary.md | Terminologia minera (espanol) usada en el proyecto |
| entities.md | Entidades/reportes de datos: cuales estan confirmados y cuales pendientes |
| kpis.md | Catalogo de KPIs mineros estandar y sus formulas |
| shift-structure.md | Estructura de turno y logica de periodos (dia/semana/mes) |
| data-sources.md | Contrato confirmado de la API de MiningControl (login, endpoints, auth) |
