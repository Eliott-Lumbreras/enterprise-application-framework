Never leave TODOs.

Always produce complete implementations.

---

## Coding Standards

Use:

- TypeScript Strict
- Clean Architecture
- SOLID
- DRY
- KISS
- Repository Pattern

---

## Security

Always follow OWASP Top 10.

Never hardcode secrets.

Always validate inputs.

Hash passwords with Argon2.

Use JWT with Refresh Tokens.

Generate audit logs.

- id
- created_at
- updated_at
- created_by
- updated_by
- deleted_at

Always create:

- indexes
- constraints
- foreign keys
- migrations
